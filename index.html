About this demo

This is a small demo storefront created to showcase a simple shopping UI and checkout flow. It is intended for learning and prototyping only.

Built with
  • HTML5, semantic markup
  • CSS3 + custom styles in index.css
  • Bootstrap 5 (CSS + JS bundle via CDN) for layout and modals
  • Tailwind Play CDN utilities for quick helpers (used lightly)
  • Vanilla JavaScript in app.js (products, cart, checkout, localStorage)
  • localStorage for persisting profile and orders (no backend)
  • Images from Unsplash / public demo assets
How to run

Open index.html in a modern browser. No build step required. For development, serve the folder with a static server (e.g., Live Server) for nicer reloads.

Notes

Payments are mocked and delivery info is dummy data. Data is stored locally in your browser's localStorage.
